# Front-End-Projects

EduFeedback — Anonymous Teacher Feedback System
A clean, fully client-side web app that lets students submit anonymous feedback for their teachers. Built with pure HTML, CSS, and vanilla JavaScript — no backend, no frameworks, no build tools required.

 Features:
Student Side:

. Department browser — choose from 7 departments (CS, Math, Physics, English, Chemistry, Biology, Psychology)
. Teacher listing — view all teachers in a department with their average rating
. Overall star rating — 1–5 stars with descriptive labels (Poor → Excellent)
.  Aspect ratings — rate Teaching Clarity, Subject Knowledge, Punctuality, Student Engagement, and Fair Assessment individually
.  Written comment — optional free-text feedback up to 500 characters
. 100% anonymous — no name, roll number, or device info is ever stored

Admin Side:

. Hidden admin panel — accessible only via #admin URL hash
. Dashboard stats — total feedbacks, teacher count, department count, average rating
. Feedback viewer — filter by department, teacher, and sort by date or rating
. Teacher management — add, edit, and delete teachers with photo support
. Clear all feedback — bulk-delete option with confirmation
